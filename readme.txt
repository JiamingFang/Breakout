This game is created by Jiaming Fang on 09/27/2018
Press A and D to move the paddle left and right
Press W and S to change the width of the paddle
Press T to start the game

Press R to restart the game
Press Q to quit the game
Press J to shoot the ball

Try to break as many brickes as you can!
The color of the bricks will become more intense as the brickes are destroyed.

Double Buffering used to reduce flickering.
